What it does, in plain English

When you search Google for a hotel, you see one nightly price. That's it. Hotel Hacker adds everything Google hides.

  • 💰 True total cost. Rolls resort fees, parking, and wifi into the math so the ranking can't lie. The number you see is the number you'll actually pay.
  • 🎯 Points & free-night smarts. Knows Marriott 5→4, IHG 4th-night-free, Hilton 5th-free, and Hyatt's category sweet spots. Tells you when paying with points beats paying cash.
  • 💎 Amex FHR perks valued in dollars. $100 F&B credit, daily breakfast for two, 4pm checkout, room upgrade — added up against the rate so you can see if FHR actually wins.
  • 💱 Currency arbitrage flags. Same property quoted in JPY vs USD can swing 2–8%. We notice. You don't have to.
  • 🚩 Risk badges on every row. LEGAL / GRAY / TOS-RISK so you always know what you're booking and what you're risking.
  • 📊 Every number explains itself. Each row shows the raw price → effective price math in one sentence. No black boxes.
  • 🔒 Local only. Your balances and searches never leave your computer. No accounts, no tracking, no cookies.

Why this is better than Google

Google Hotels is fine for a quick nightly rate. It is blind to almost every way real travelers actually save money — points, free-night mechanics, Amex perks, and currency mismatches. Hotel Hacker is the layer Google doesn't see.

How to install (10 minutes, no coding required)

If you've never opened Terminal before, that's okay — we'll walk you through it. Every step explains what's about to happen and why, before any command.

What you need first

  • A Mac or Linux computer. (Windows users: see the bottom of this page — we'll walk you through WSL.)
  • About 10 minutes.
  • A FREE SerpApi account at serpapi.com. The free tier covers casual use; no paid plan required. (SerpApi is the service that lets the tool see hotel prices.)
  • (Optional) An Amex Platinum, Centurion, or Business Platinum card — only if you want to compare Fine Hotels & Resorts perks.

  1. Open Terminal (the black window where you type commands)

    If you've never seen Terminal before, that's okay — it just looks like a black box with a blinking cursor. On a Mac: press Cmd + Space, type "Terminal", press Enter. A black window opens. That's where you'll paste the commands below.

  2. Check that Python is installed

    Python is the language Hotel Hacker is written in. Most Macs already have it. Copy-paste this into Terminal and press Enter:

    python3 --version

    If you see Python 3.10 or higher, you're good. If not, download from python.org/downloads first, then come back.

  3. Download the tool

    This just copies the tool to a folder on your Desktop. It takes about 10 seconds. Paste this whole block (two commands joined by &&):

    git clone https://github.com/joseantoniopau/Hotel-Hacker ~/Desktop/hotel-hacker && cd ~/Desktop/hotel-hacker

    You'll end up inside a new folder called hotel-hacker on your Desktop.

  4. Run the installer

    This sets up the helper files Hotel Hacker needs to run. Takes about 30 seconds.

    ./install.sh

    You'll see lots of words go by — that's normal, it's just listing each helper as it gets installed. When it finishes, you'll see a "done" message and your normal cursor returns.

  5. Add your free login key (also called an API key)

    After you sign up at serpapi.com, they'll give you a key — it'll look like a long string of letters and numbers. This step lets the tool use that key to look up hotel prices on your behalf.

    ./setup-keys.sh

    When it asks for the SerpApi key, paste it in and press Enter. That's it.

  6. Tell it your points balances

    This is what makes the tool know which redemptions you can actually book. Open this file with TextEdit (any plain text editor works):

    ~/Desktop/hotel-hacker/data/user_balances.json

    You'll see a list of programs with zeros next to them. Change the zeros to your actual point balances (log into your card and hotel accounts to check). Save the file. If you don't have any points yet, just leave the zeros — the tool still works for cash and FHR comparisons.

  7. Launch the tool

    This starts the tool and serves the website on your own computer.

    python3 ui/server.py

    You'll see Uvicorn running on http://127.0.0.1:8788. Leave this Terminal window open — that's the tool running. If you close it, the website goes down.

  8. Open the website

    Open your web browser and go to:

    http://127.0.0.1:8788

    The Hotel Hacker interface loads. You're done.

How to use it

  1. Type your destination in the search box.

    Like "Tokyo" or "Park Hyatt Marrakech". Be as specific as you want.

  2. Pick your check-in and check-out dates.

    Click the date picker, choose the two dates, done.

  3. Click "FIND HOTELS".

    Big amber button on the right side of the search bar.

  4. Wait about 10–20 seconds.

    The tool is talking to SerpApi and crunching points math in the background. First search for a route is slower; later ones are cached and instant.

  5. Look at the EFFECTIVE column.

    That's the true out-the-door cost — whether you pay cash, use points, or book through FHR. Sorted cheapest first. Hover any row to see the one-sentence math.

  6. Click "Book" on the row you want.

    You'll be taken directly to the hotel's official site (or Amex Travel for FHR). You book there, with your own card. The tool never touches your payment info.

Things you can also do

  • Paste an FHR rate manually. If you're on amextravel.com and see a Fine Hotels & Resorts rate for your stay, paste it into the FHR box on any row. The tool re-ranks instantly with the FHR perks valued in dollars.
  • Draft a polite upgrade-request email. Click "Draft email" on any row. The tool writes a courteous pre-arrival message — you copy it into your own email and send it. (We never send anything for you.)
  • Update your balances anytime. Re-open data/user_balances.json, change the numbers, save. The tool picks it up on the next search.
  • Watch the SerpApi quota counter. The small number in the corner shows how many free searches you have left this month. It refreshes lazily — no surprises.

Tomorrow and after

You only install once. After that, to use it again:

  1. Open Terminal
  2. Type: cd ~/Desktop/hotel-hacker && python3 ui/server.py
  3. Open http://127.0.0.1:8788 in your browser

To stop it: in the Terminal window, press Ctrl + C.

Privacy and safety

  • ✅ Runs on your own computer. Nothing is uploaded anywhere. The tool just makes API calls to SerpApi (like your browser does when you visit a site).
  • ✅ We never see your Amex card. FHR perks are valued from a rate YOU paste in from amextravel.com. The tool has no card-on-file, no Amex login, nothing.
  • ✅ No tracking, no ads, no cookies. Unlike Google, your searches are never tied to your identity.
  • ✅ Bookings happen on the hotel's or Amex Travel's site. Hotel Hacker just points you there — it never handles your payment info.
  • ✅ We can draft a polite pre-arrival upgrade-request email, but we never send it. You copy/paste it into your own email and send it yourself. Always your call.
⚠️ TOS-RISK tactics: A handful of tactics (VPN/locale-shopping to get different regional prices) are shown with a red TOS-RISK badge and full warnings. We never auto-execute them. You decide what's right for you.

Troubleshooting

I get an error running ./install.sh
This is almost always a Python version issue. Type python3 --version — if it's below 3.10, install a newer Python from python.org/downloads and try again.
The website doesn't load at 127.0.0.1:8788
The tool only runs while the Terminal window from step 7 is open. If you closed it, just reopen Terminal and run cd ~/Desktop/hotel-hacker && python3 ui/server.py again.
No hotels are coming back when I search
Check that your SerpApi key was saved correctly. In Terminal: cat ~/Desktop/hotel-hacker/.env. You should see a line starting with SERPAPI_KEY= followed by your key. If it's empty or missing, run ./setup-keys.sh again.
I changed my balances but the tool doesn't use them
Make sure you saved the file (Cmd + S in TextEdit) and that you edited the right file: ~/Desktop/hotel-hacker/data/user_balances.json. Refresh the web page. If TextEdit added a .txt extension by accident, rename the file back to user_balances.json.
I can't figure out where to find the FHR rate to paste in
On amextravel.com, search for your hotel and dates. Look for the "Fine Hotels & Resorts" filter or badge. The rate you want is the "best flexible" nightly rate — the one that doesn't say "non-refundable". Paste that USD number into the FHR box on the matching row in Hotel Hacker.
I'm on Windows
Install WSL (Windows Subsystem for Linux) using Microsoft's instructions. Once WSL is set up, open the Ubuntu/WSL terminal and follow the Mac steps from there — they all work identically inside WSL.
The SerpApi quota counter says I'm running out
The free tier is 250 searches/month. Searches you've already done in the last 24 hours are cached locally and don't count again. If you genuinely need more, SerpApi has paid tiers — but most casual users never come close.

What this does NOT do

  • ❌ Does not book the hotel for you. It shows you the cheapest path and links you to the official site (or Amex Travel for FHR). You book there.
  • ❌ Does not see your credit card. You only paste in a rate number from amextravel.com. No card details, no logins, nothing leaves your machine.
  • ❌ Does not track you. Everything is local. No accounts, no cookies, no analytics, no phone-home.
  • ❌ Does not help with flights. For flights, use the sibling project: flight_hacker (and its companion site joseantoniopau.github.io/flight_hacker).
